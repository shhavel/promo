# Promo REST client

Wrapper for promo REST server
Very basic REST client for Marketing Opt-Ins App

## Installation

Add this line to your application's Gemfile:

    gem 'promo'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install promo
